Course Overview

What is Hardware
& Networking?

This diploma covers computer hardware components, assembly, troubleshooting and network infrastructure. You'll learn to set up, maintain and secure computer networks used in businesses and organisations.

With hands-on lab sessions, you'll gain practical experience in assembling PCs, configuring routers, setting up LAN/WAN networks and troubleshooting hardware and software issues.

Syllabus

What You Will Learn

From computer assembly to advanced network configuration and security.

MODULE 01

Computer Hardware

  • PC components & peripherals
  • Motherboard, CPU & RAM
  • Storage devices & ports
  • PC assembly & disassembly
MODULE 02

Operating Systems

  • Windows installation & setup
  • Linux basics
  • OS troubleshooting
  • Drivers & updates
MODULE 03

Networking Basics

  • Network types & topologies
  • IP addressing & subnetting
  • OSI & TCP/IP model
  • Network cables & connectors
MODULE 04

Network Devices

  • Routers & switches
  • Hub, modem & access points
  • VLAN configuration
  • Wireless networking
MODULE 05

Network Security

  • Firewall setup & configuration
  • VPN & remote access
  • Network threats & prevention
  • Antivirus & security tools
MODULE 06

Troubleshooting & Support

  • Hardware fault diagnosis
  • Network troubleshooting
  • Help desk & IT support skills
  • Documentation & reporting
