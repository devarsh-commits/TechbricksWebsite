What You Will Learn

A structured curriculum covering all essential computer skills needed in today's world.

Computer Basics

  • Introduction to computers & history
  • Hardware components & peripherals
  • Input & output devices
  • Computer memory & storage
MODULE 02

Operating System

  • Introduction to Windows OS
  • File & folder management
  • Control panel & settings
  • Keyboard shortcuts & productivity
MODULE 03

MS Office – Word

  • Tables, headers & footers
